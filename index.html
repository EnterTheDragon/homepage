This page features Frontend work that I have made either by myself or in a collaboration with designers over the past few years.

SportsShoes.com Global Nav

This project was a complete redesign of the navigation which covers almost the entire SportsShoes site.

The task required close working with designers and the external UI agency to bring the plans to life.

Features:

  • Responsive
  • Database driven
  • JavaScript and CSS animations
  • Srcset responsive images

Under Armour Run Fast Run long

This feature store on SportShoes.com is advertising Under Armour's latest range of running shoes.

Users can find shoes on the page in a slider and can find shoes using an interactive video shoe finder.

Features:

  • Responsive
  • Mobile first
  • JavaScript and CSS animations
  • Interactive videos

New Balance Fresh Foam

A site made in 2020 to advertise New Balance's extensive range of Fresh Foam running shoes.

The carousel features all the models of the range and also allows users to select different colours of the shoe.

Features:

  • Responsive
  • Mobile first
  • JavaScript and CSS animations
  • HTML data changes in JavaScript
